Creating classes

The concept of extending classes using basic inheritance to alter behaviors within child classes. Inheritance in JavaScript is based around the prototype object.

All objects that are built from the prototype share the same functionality. When you need to code more complex OOP relationships, you can use the class keyword and its easy-to-understand and easy-to-reason-about syntax.
